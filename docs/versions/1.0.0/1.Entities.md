# Prepare the entities
- first thing in this project, I though about entities. I need to data in the database to be able to work with it.
- I decided to create a EntityExtensionsBundle because here I will add logic that is reusable across all entities. For example, I started with the translations : We will use interfaces and traits (I choose these because by creating a interface, I can resuse it across the project and if you tag your entity as my interface, it basically is the same thing as my interface and you can use all the helper methods/classes I build. I used traits so you can add them easly in your class and it doesn't block you to extend only once from my class)
